# 设计文档
本项目从互联网网站采集A股股票信息，进行数据加工、存储,然后基于加工后的数据进行数据分析。

本文档描述了本项目数据处理流程、数据结构、数据分析等相关设计。


## 数据处理流程

```
爬虫抓取网页数据 -> json格式写入到文件中 -> 从文件导入到数据库中 -> 数据分析 -> 数据展示 
```
